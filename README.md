# Innorbia · Sitio web

Sitio web de una sola página (landing) para **Innorbia — Consultora Integral de Ingeniería de Producción**.
Estático (HTML + CSS + JavaScript), sin dependencias ni backend: listo para publicar en **GitHub Pages**.

> *Transformamos procesos, impulsamos el crecimiento.*

## ✨ Características

- Diseño corporativo-industrial limpio, profesional y elegante con la paleta del logo (azul + verde).
- Animación de **Metodología ÓRBITA®** como sistema orbital interactivo (6 fases).
- Animaciones de aparición al hacer scroll, contadores, barra de progreso y micro-interacciones.
- Tipografía: Archivo + IBM Plex Sans + IBM Plex Mono (Google Fonts).
- 100 % responsivo (escritorio, tablet y móvil) y accesible.
- Formulario de contacto que abre el cliente de correo (sin servidor).

## 📁 Estructura

```
innorbia-web/
├── index.html      # Página principal (punto de entrada)
├── styles.css      # Estilos, paleta y animaciones
├── script.js       # Interacciones (nav, reveals, órbita, formulario)
├── assets/         # Logo e imágenes del equipo
├── .nojekyll       # Indica a GitHub Pages que sirva los archivos tal cual
└── README.md
```

> Todo el proyecto es autocontenido: las imágenes están en `assets/` con rutas relativas.
> Lo único que se carga por internet son las tipografías de Google Fonts (estándar en GitHub Pages).

## 🚀 Publicar en GitHub Pages

1. Crea un repositorio nuevo en GitHub y sube estos archivos (que `index.html` quede en la raíz).
2. Ve a **Settings → Pages**.
3. En *Source* elige la rama `main` y la carpeta `/ (root)`. Guarda.
4. En unos segundos tu sitio estará en `https://<usuario>.github.io/<repositorio>/`.
